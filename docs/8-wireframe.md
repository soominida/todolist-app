## 공통 규칙

- 비인증 사용자가 WF-03 ~ WF-05에 접근하면 WF-01로 리다이렉트 (NFR-01)
- 모든 API 요청에는 JWT Bearer 토큰 포함 (FR-U02 이후)
- 에러 메시지는 해당 입력 필드 하단에 인라인 표시
- 모바일 기준 너비: 40자 / 데스크톱 기준 너비: 80자

## 상세 결과

### SCN-01: 신규 사용자 회원가입 및 첫 로그인
- 테스트 계정: `prod_e2e@test.com` / `Test1234`
- 회원가입 성공 화면("회원가입이 완료되었습니다.") 정상 표시
- 로그인 후 할일 목록(`/`) 리다이렉트 확인
- 헤더에 이메일 표시 확인

### SCN-02: 카테고리 및 할일 생성
- "업무", "개인" 카테고리 생성 완료
- "프로젝트 기획서 작성" (업무, 2026-05-10) → 미완료 뱃지 정상 표시
- "운동 계획 세우기" (개인, 2026-04-01) → **기한초과** 뱃지 정상 표시

### SCN-03: 카테고리 필터 및 상태 필터
- `?categoryId=1` → 업무 카테고리 할일만 표시 확인
- `?status=overdue` → 기한초과 할일만 표시 확인

### SCN-04: 할일 완료 처리
- "프로젝트 기획서 작성" 완료 처리 → "완료" 뱃지 및 녹색 체크 정상 표시
- "완료 취소" 버튼으로 전환됨 확인

### SCN-05: 기한초과 할일 수정
- "운동 계획 세우기" 종료일을 `2026-06-30`으로 변경
- 상태가 "기한초과" → "미완료"로 정상 변경

### SCN-06: 카테고리 삭제 시 미분류 처리
- "개인" 카테고리 삭제
- "운동 계획 세우기"의 카테고리가 "개인" → "미분류"로 정상 이동
- 카테고리 드롭다운에서 "개인" 항목 제거 확인

### SCN-07: 할일 삭제
- "프로젝트 기획서 작성" 삭제 확인 모달 → 삭제 완료
- 목록에서 제거됨 확인

### SCN-08: 로그아웃 및 비인증 접근 차단
- 로그아웃 버튼 클릭 → `/login` 리다이렉트 확인
- 비인증 상태에서 `/` 접근 → `/login` 리다이렉트 확인
- 비인증 상태에서 `/categories` 접근 → `/login` 리다이렉트 확인
